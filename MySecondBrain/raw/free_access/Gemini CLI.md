# Gemini CLI — полный аналог Claude Code бесплатно

## Ситуация/Проблема
Claude Code платный, нужен бесплатный AI-агент с похожим функционалом

## Решение
Gemini CLI — бесплатный CLI-инструмент от Google с 2M токенов контекстом и встроенной интеграцией с Google Workspace.

## Преимущества
- **2,000,000 токенов** контекст (vs 200,000 у Claude)
- Бесплатный тариф: 1000 запросов/день
- Встроенный доступ к Gmail, Google Docs, Drive
- Telegram бот для удобного взаимодействия

## Установка на сервере
1. Купить сервер (USA, Ubuntu 24.04) — ~$1/мес
2. `curl https://ollama.ai/install.sh | sh`
3. `npm install -g @google/gemini-cli`
4. Авторизоваться через Google аккаунт
5. Скачать файлы настройки из Telegram канала автора

## Файлы настройки (12 штук для полной настройки)
1. Окружение и установка
2. Node.js 22
3. Gemini CLI
4. Авторизация
5. Структура папок
6. Telegram Bridge
7. Мультимодальность (фото, голос, документы)
8. Память (3 слоя: SQLite, агент, Obsidian)
9. Продвинутая архитектура + сжатие контекста
10. Инструменты (Firecrawl, n8n, Google Workspace)
11. Планировщик задач
12. Система команд

## Подключение Telegram
1. Создать бота через @BotFather
2. Получить API токен
3. Узнать свой Chat ID через @userinfobot
4. Настроить bridge внутри Gemini CLI
5. Команды бота появятся автоматически

## Модели
- **Gemini 3.1 Pro** — бесплатно, 1000 запросов/день
- Автоматическое переключение между моделями
