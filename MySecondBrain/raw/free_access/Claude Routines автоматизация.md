# Claude Routines — автоматизации 24/7

## Ситуация/Проблема
Нужно автоматизировать рутинные задачи (утренние дайджесты, мониторинг, отчеты) без ручного запуска

## Решение
Claude Routines — автоматизации, работающие на серверах Anthropic. Компьютер может быть выключен, но задачи выполнятся.

## Триггеры
1. **По расписанию** — ежедневно, еженедельно, по cron
2. **По API** — при внешнем запросе
3. **По событиям GitHub** — PR, релизы и т.д.

## Лимиты (включены в подписку Claude)
| Тариф | Запусков/сутки |
|-------|---------------|
| Pro | 5 |
| Max | 15 |
| Team/Enterprise | 25 |

## Настройка (пошагово)
1. Скачать Claude Code Desktop
2. Подключить GitHub (Settings → Connectors → GitHub)
3. Создать GitHub репозиторий (приватный)
4. Создать Routine → Remote (не Local!)
5. Выбрать репозиторий, написать промт с задачей
6. Настроить Environment (TG bot token, API keys)
7. Выбрать White List домены (custom)
8. Настроить Permissions (рекомендуется — не давать права на main branch)
9. Добавить триггер (время/GitHub event/API)
10. Тестировать через "Run Now"

## Пример: Утренний дайджест Gmail → Telegram
- Gmail коннектор забирает непрочитанные письма
- Claude анализирует и формирует краткий отчет
- Telegram бот отправляет результат

## Пример: AI дайджест из YouTube, GitHub, Telegram
- Парсит YouTube по запросам (AI, Claude, etc.)
- Собирает горячие GitHub репозитории
- Мониторит Telegram каналы
- Формирует единый отчет в Telegram

## Environment Variables (безопасность)
Никогда не писать API ключи прямо в промт! Только в Environment:
```
TG_BOT_TOKEN=...
TG_CHAT_ID=...
YOUTUBE_API_KEY=...
```
