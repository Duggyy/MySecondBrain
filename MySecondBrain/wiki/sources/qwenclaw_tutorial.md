## Описание

Оригинальный туториал по установке QwenClaw — бесплатного AI-агента на базе Qwen модели. На момент публикации работал бесплатно. **Лимиты срезаны на следующий день после выхода видео.**

## Ключевые возможности (на момент публикации)

- Бесплатная модель Qwen (1000 запросов/сутки)
- Telegram бот как интерфейс
- Cron tasks / напоминания
- Code writing на сервере
- Работа с файлами
- Интеграция с Google Docs, Яндекс.Диск

## История
