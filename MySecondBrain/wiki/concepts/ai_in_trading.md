## Обзор

AI трансформирует трейдинг: от анализа графиков до автоматического написания стратегий.

## Ключевые применения

1. **Автоматизация рутины** — подготовка, разбор результатов
2. **Анализ графиков** — AI видит паттерны
3. **Генерация стратегий** — Pine Script через TradingView API
4. **Управление рисками** — мониторинг позиций

## Инструменты

| Инструмент | Назначение |
|------------|-----------|
| Claude Code + TradingView MCP | Полный цикл: анализ → стратегия → тест |
| BitGet MCP | Автоматическое исполнение сделок |
| Pine Script | Язык для TradingView стратегий |

## Ограничения

- AI не предсказывает рынок — он оптимизирует процесс
- Требуется человеческий контроль
- Юридические вопросы автоматической торговли
