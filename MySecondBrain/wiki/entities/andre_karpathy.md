## Кто такой

Andre Karpathy — исследователь в области AI, один из основателей OpenAI. Бывший директор Computer Science в Tesla. Известен своими образовательными материалами по deep learning и LLM.

## Ключевые идеи

- **LLM Wiki** — паттерн построения персональной базы знаний с помощью LLM
- **Stats for LMDD** — микро-библиотека для работы с языковыми моделями

## Вклад в AI-сообщество

- Создал cs231n (Stanford CNN course)
- Разработал микро-LLM библиотеки (llmtime, llmdd)
- Популяризировал использование Claude Code + Obsidian для memory systems
