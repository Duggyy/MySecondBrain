## Summary

Практический гайд по использованию Claude Code как автономного агента для полной настройки личного VPS-сервера и развертывания защищённого VPN. Нейросеть сама создаёт SSH-ключи, настраивает firewall, Fail2Ban и развертывает 3X-UI с VLESS Reality.

## Принцип

**От чатов к агентам**: Claude Code это не просто чат в браузере — полноценный агент, который может выполнять реальную работу локально на вашем ПК и удалённо на сервере.

## Проблема & Решение

**Проблема**: 
- ChatGPT, Gemini, Claude часто не работают в РФ/БЛР
- Готовые VPN медленные или заблокированы
- В худших случаях сливают данные
- Терминал и команды кажутся непонятными

**Решение**: 
- Свой личный VPS (~$5/месяц)
- Настроить VPN с защитой от блокировок
- **Claude Code делает 99% работы**

## Требования

### Claude Pro (20$ в месяц или $100-200 за максимальные версии)
- Бесплатная версия недостаточна
- Нужна virtual phone number (для регистрации)
- Нужна virtual card (для оплаты)

### VPS-сервер
- **ОС**: Ubuntu 22.04
- **Локация**: Европа (вне блокировки)
- **Конфиг**: 1 ядро, 512MB RAM достаточно для VPN
- **Скорость**: обратить внимание на линк
- **Примеры провайдеров**: OVH, и др. (список в Telegram автора)

## Процесс Настройки

### 1. Подготовка (Minimal Terminal Commands)
```bash
ssh ubuntu@<server_ip>
# Ввести пароль из письма
# Терминал попросит изменить пароль
# Устанавливаем новый пароль
```

Это **всё, что нужно делать вручную**. Дальше Claude Code.

### 2. Настройка Claude Code на сервере
- Загружаем агента на VPS
- Даём ему "скилл" для VPN настройки

### 3. Автоматическое Hardening Сервера (Claude Code делает всё)
- Создание SSH-ключей
- Настройка Firewall
- Установка Fail2Ban (защита от brute-force)

### 4. Развертывание VPN (Claude Code делает всё)
- Установка 3X-UI панели
- Настройка VLESS Reality протокола
- Конфигурация маскировки под реальные сайты

## Почему VLESS Reality?

- **Один из немногих протоколов**, который сложно блокируется в РФ
- **Маскировка под трафик к сайтам** — выглядит как обычный HTTP
- **Устойчив к блокировкам** — не отслеживается как VPN
- Альтернативы: Amnesia, Hysteria 2 (тоже хорошие варианты)
